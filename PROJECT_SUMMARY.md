## 🔧 技术栈

- **前端框架**: Vue3 + Composition API
- **跨平台**: uni-app
- **构建工具**: Vite
- **UI框架**: uview-plus
- **状态管理**: 简单全局状态管理（Pinia备用）
- **网络请求**: uni.request封装
- **数据可视化**: uCharts
- **代码规范**: ESLint + Prettier
